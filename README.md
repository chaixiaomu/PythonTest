## 学习 python 的 demo 的集合

创建本地仓库：
git init

将所有的文件提交到本地仓库：
git add .
git commit -m ‘提交信息’

将本地的仓库关联到远程仓库：
git remote add origin https://github.com/chaixiaomu/PythonTest.git
git pull origin master

查看一下工作区是否已经干净
git status
On branch master
nothing to commit, working tree clean

提交到远程仓库
git push -u origin master
